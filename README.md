# jar-express-base-api

Proyecto de aprendizaje para crear una api en Express sin seguridad

## Arquitectura

API Rest Modular (Screaming Arquitecture) con Patrón Repositorio y validación basada en esquemas.

## Stack tecnológico

- [x] TypeScript
- [x] Dependencias: Express, Zod, Dotenv, Swagger, Helmet
- [x] Linters: ESLint, Prettier
- [x] Cliente para peticiones: Bruno
- [x] Script backup base de datos
- [x] Swagger
- [x] Vitest + Supertest
